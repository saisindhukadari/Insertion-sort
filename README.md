# Insertion Sort

## Description

Insertion Sort is a comparison-based sorting algorithm that builds the sorted portion of the array one element at a time. It takes each element from the unsorted portion and inserts it into its correct position within the sorted portion by shifting larger elements to the right.

## Algorithm

1. Start from the second element of the array.
2. Consider the current element as the key.
3. Compare the key with the elements before it.
4. Shift all elements that are greater than the key one position to the right.
5. Insert the key into its correct position.
6. Repeat the process until the entire array is sorted.

## Time Complexity

* **Best Case:** O(n)
* **Average Case:** O(n²)
* **Worst Case:** O(n²)

## Space Complexity

* **O(1)**
